# What I've learned

**bang operator**
- ! is the bang operator that takes the opposite of whats in from of it
- !true = false
- !false = true

```
var gorden = false; // gorden is false

!gorden; // gorden is opposite of false = true

gorden = !gorden; // = true
// gorden is false = gorden is opposite of false

gorden; // = true
```
